Recent Projects

Project 1

Shopify Sales Data Analysis

In this Shopify Sales Data Analysis project, I built an interactive Power BI dashboard to uncover key insights into transaction performance, customer purchasing patterns, and long-term value using a comprehensive Shopify sales dataset. The analysis focused on KPIs such as net sales, average order value, repeat customers, lifetime value, retention rate, and purchase frequency.

Project 2

Credit Risk Classification App

In this Credit Risk Classification App project, I developed and deployed an interactive web application using Python, XGBoost, and Streamlit to predict whether credit applicants are low-risk ("good") or high-risk ("bad") based on their profiles. The model was trained on the classic German Credit Data dataset, where XGBoost was selected for its superior accuracy after comparing multiple algorithms.

Project 3

Insurance Payment Prediction

In this Insurance Payment Prediction project, I built and deployed a Random Forest Regressor to predict medical insurance charges from features like age, BMI, smoking status, and region using the Medical Cost dataset (1,338 records). After EDA, feature engineering, and model comparison/tuning in a Jupyter notebook, the optimized model was deployed as an interactive Streamlit app with Docker support for instant predictions. This end-to-end solution supports accurate premium setting, risk assessment, and actuarial planning while showcasing reproducible ML deployment skills.

Project 4

Booking.com Hotel Scraper

In this Booking.com Hotel Scraper project, I built a Python web scraping tool using BeautifulSoup, Requests, and lxml to automatically extract hotel details from any given Booking.com search URL for a city and date range. The script prompts users for input, fetches the page with polite delays and headers to avoid blocks, parses the HTML cleanly, and saves structured results directly to a local CSV file for easy analysis or use. This automation enables efficient collection of travel data for market research, price comparison, or data-driven travel planning.

Project 5

End-to-End Medical Chatbot

In this End-to-End Medical Chatbot project, I built a context-aware RAG (Retrieval-Augmented Generation) application using LangChain, OpenAI GPT models, sentence-transformers embeddings, and Pinecone vector database to ingest and query medical PDFs for accurate, evidence-based responses. The system processes documents with pypdf, stores semantic vectors in Pinecone, and delivers interactive conversations via a Flask web interface, with environment management via dotenv. Fully containerized with Docker and deployed on AWS (EC2 + ECR) using GitHub Actions CI/CD, it demonstrates scalable LLM integration, vector search, web development, and cloud deployment.
